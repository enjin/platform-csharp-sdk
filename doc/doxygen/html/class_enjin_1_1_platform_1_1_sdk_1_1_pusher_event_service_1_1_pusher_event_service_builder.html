InvalidOperationExceptionThrown if the key is null or if both the host and cluster are set at the time this method is called.

If host is not set, then the service will default to ws-{cluster}.pusher.com and if the cluster is not set, then the cluster will default to mt1. If encryption is not set, then encryption will default to true.

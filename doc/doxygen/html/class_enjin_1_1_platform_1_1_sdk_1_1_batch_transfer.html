Detailed Description

A request to act as a mutation for transferring multiple tokens in one transaction.

Up to 250 different transfers per batch may be included.

The continueOnFailure flag may be used to allow all valid transfers to complete while skipping transfers which would fail so that they may be fixed and attempted again in another transaction.

Returns the string representing this range.

Returns
The string representing this range.

If Start and End are not equal, then the returned string will be the starting and ending values separated by two periods. For example, if the starting value is 3 and the ending value is 8, then the string will be returned as 3..8.

If the values are equal, then the returned string will be the string representing the shared value.
